Part 2. [발표회] 교육생 개발 AI Agent 서비스 공유

참가자들이 AI 에이전트(Antigravity 등)와 주도적으로 소통하여 본인의 필요에 의해 직접 기획하고 개발한 5가지의 서비스 사례입니다.

🔮 타로 상담 어플 (대화형 타로 테라피) 김*리 님

💡 기획 배경

  • 2030 세대가 사주/타로 어플을 많이 사용한다는 트렌드에 착안.
  • 기존 어플들은 정해진 단발성 질문만 제공하는 한계가 있어, AI와의 유연한 상담 대화를 통해 마음속 진짜 고민을 구체화하고 맞춤형 해결책을 도출하도록 설계함.

⚙️ 주요 기능

  • 턴제 상담 프로세스: 사용자가 3번 대화를 주고받는 동안 자연스럽게 고민을 명확화하도록 구현.
  • 타로 카드 매칭: AI 이미지 생성(나도바나나)을 거친 고유 타로 카드 3장을 임의 선택하고 이에 따른 운세 해설 및 '오늘의 할 일 처방전' 매칭.
  • 결과 이미지 다운로드: 타로 해설 결과를 처방전 형태의 이미지로 로컬에 바로 저장하는 기능.

🌱 어려웠던 점 & 깨달음

  • 원하는 방향 유도의 어려움: AI가 지시와 다른 답변을 할 때 프롬프트를 다시 지시하고 정교화하는 방향 튜닝에 공수가 필요했음.
  • 토큰의 가치: 교육 담당자가 제공한 토큰 지원 초대 링크 덕분에 부족함 없이 풍성한 실습이 가능했음.
  • API 보안 및 과금 인지: API 키 연동을 통해 고도화된 기능이 가능해졌으나, GitHub 공개 저장소 업로드 시 보안 위협을 깨닫고 외부 독립 도메인 배포를 택함. 또한 사용량에 따른 과금 비용 발생 메커니즘을 경험함.
📊 아시아 영업 모니터링 & 리스크 관리 대시보드 신*중 님

💡 기획 배경

  • 아시아 지역 가전 영업 담당자로서 지정학적 리스크, 뉴스, 기후 변동 등이 가전제품 판매와 물류에 미치는 영향을 사전에 실시간 파악하고 대응하기 위해 제작.

⚙️ 주요 기능

  • 리스크 지표 자동 분류: 관련 시장 정보 및 뉴스를 영향도(심각한 리스크, 기회 요인 등)에 따라 실시간 분류하고 국가별 발생 현황 시각화.
  • 물류 지연 리스크 연동: 주요 항만의 선박 체류 시간 등 물류 데이터를 API를 통해 실시간 연동하여 모니터링.
  • 정보 검색 연계: 현지 규제 변화 및 경쟁사 동향 뉴스를 연동하고, 상세 확인이 가능하도록 구글 검색과 연계.
  • UI 디자인 개선: 투박하게 생성된 초기 지도 국경선을 안티그래비티와의 반복된 피드백 소통으로 수려한 그래픽 레이아웃으로 변경.

🌱 어려웠던 점 & 깨달음

  • 실시간 데이터 연동의 효용: 단순 정적 엑셀 정리를 넘어 실시간 정보를 받아오는 코드를 연결하며 진정한 '에이전트' 협업의 즐거움을 느낌.
  • 향후 계획: 서비스 완성도를 높여 업무에 적극 적용하고, 차후 광고 연동이나 타 비즈니스용 앱 개발도 구상 중임.
🗺️ 가성비 문화생활 정보 & 캘린더 연동 서비스 박*혜 님

💡 기획 배경

  • 무료 혹은 가성비 높은 공연, 전시 등 공공 문화 행사 정보를 쉽게 찾고 일정에 연동하고 싶어 기획. (정적 웹 환경의 한계로 초기 식물 나눔 서비스에서 피보팅함)

⚙️ 주요 기능

  • 공공 API 연동: 공공데이터포털, 서울 열린데이터 광장, 문체부 등의 데이터를 기반으로 무료/저렴한 전시 정보 수집.
  • 다중 뷰 레이아웃: 전시 행사를 리스트형, 카드형, 지도형(Map) UI로 직관적으로 제공.
  • 일정 등록 연동: 원하는 문화 정보를 바로 개인 구글 캘린더에 바로 등록할 수 있는 바로가기 구현.

🌱 어려웠던 점 & 깨달음

  • 배포 환경에서의 API 오작동: 로컬 PC에서는 정상 가동되던 API 키가 깃허브 원격 서버로 배포 시 작동하지 않는 에러를 마주하며, 도메인 등록 유무 및 보안 정책에 따른 키 연동 차이를 명확히 체감함.
  • 리소스 한계: 실습 도중 토큰이 조기 소진되어 하루 단위 개발 진행 속도가 다소 더뎌 아쉬움으로 남음.
🥠 AI 포춘쿠키 & 로또 번호 추천 서비스 김*정 님

💡 기획 배경

  • 사무실 내 동료들이 가볍게 참여하여 미소 짓고 힐링할 수 있는 소소한 사내 엔터테인먼트 어플 기획.

⚙️ 주요 기능

  • 인터랙티브 모션: 포춘쿠키가 쪼개지는 그래픽 효과와 함께 AI가 생성한 유쾌한 응원 운세 및 재미용 추천 로또 번호 출력.
  • 사내 테스트: 옆자리 동료에게 즉석 모바일 사용 테스트를 거쳐 긍정적인 웃음과 평가를 이끌어냄.
  • 자동 문서화: 안티그래비티에게 기획서 및 깃허브 설명 문서(README.md) 작성을 요청해 함께 배포함.

🌱 어려웠던 점 & 깨달음

  • 반응형 레이아웃 조정: 모바일에서는 온전하지만 PC 모니터 화면에 맞추는 세부 레이아웃 정렬은 초보자가 완벽히 다루기에는 난이도가 높아 적절한 시점에서 타협하고 마무리함.
  • 에이전트 중심 배포: 깃허브 배포 전 과정의 핵심 기술 지시를 안티그래비티에게 말 한마디로 시켜 해결한 것이 신선한 경험이었음. 다음 기획 시에는 기획서(아키텍처 설계)를 먼저 작성하고 빌드하는 정석을 따를 예정.
💰 경제적 자유(파이어족) 달성 자산 시뮬레이터 김* 님

💡 기획 배경

  • 컴퓨터 비전공자로서 평생 살아가는 동안 노후 및 은퇴 이후에 실제로 필요한 생활 자금이 얼마인지 냉정하게 자체 계산하고 인생 설계를 하기 위함.

⚙️ 주요 기능

  • 자산 시뮬레이션: 연령, 은퇴 시점, 필요 생활비, 현재 자산, 물가 상승률 및 투자 기대 수익률을 변수로 한 종합 계산기.
  • 소진 경고 판정: 보유 원금이 바닥나는 나이를 계산하여 경고 문구 및 시각적 안내 메시지 출력.
  • 데이터 표 및 차트화: 연도별 자산 유입/유출 추이를 그래프와 테이블 형태로 가독성 높게 표시.

🌱 어려웠던 점 & 깨달음

  • 비전공자 장벽 극복: 개발 용어나 원리를 이해하기 매우 어려웠으나, 제공된 주차별 요약 가이드와 라이브 세션 다시보기 영상을 배속으로 복습하며 끝내 구현에 성공함.
  • 개발 프로세스 및 소통력 이해: 이전에는 사내 개발자들의 산출물을 일방적으로 받기만 했으나, 직접 해보면서 기술 구현의 수고를 알게 됨. 파이썬이나 코드 구조를 더 익혀 소소한 버그는 직접 수정하고 개발팀과 대화 시 주도적인 피드백을 주고 싶어짐.

Part 4. [교육 담당자 메시지] 학습모임의 성공 기준과 미래 인재상

학습모임을 마무리하며 교육담당자가 전한 기획 소회와 앞으로의 비전 메시지 요약입니다.

1. 학습모임의 성공과 실패 기준

본 학습모임의 성공과 실패의 기로는 참가자들의 인식 및 도구 활용 태도의 변화에 있었습니다.

🟢 성공의 기준 (Success)

  • 자신감의 획득: AI로 단순 채팅만 던지던 모습에서 탈피해, 필요한 서비스나 솔루션을 내 손으로 직접 만들 수 있다는 강력한 자신감이 생겼는가?
  • 자율적 도구 탐색 의지: 정규 모임 세션이 끝난 뒤에도 현업 및 생활 속에서 능동적으로 새로운 AI 도구를 발굴해 테스트해 보려는 동기가 유발되었는가?
  • 동반 협업 파트너 인식: AI를 단순히 기계적인 응답기(검색 챗봇)가 아닌, 상호 아이디어를 주고받으며 함께 일하는 '협업 가능한 조수'로 인정하는가?

🔴 경계했던 실패의 기준 (Failure)

  • "난 결국 코더가 아니니 못해"라는 단념: 세션 동안 흥미는 있었으나, "코딩 지식이 없으니 내 실무에 내가 직접 만드는 건 역시 무리"라며 선을 긋고 예전 상태로 회귀하는 태도.
  • 단순 문답 검색의 수준 정체: 에이전트식의 복합 협업 도구를 뒤로하고, 기존 방식의 기본 웹 검색 대용품으로만 AI를 축소해 활용하는 행태.
  • 단순 예제 복제에 만족: 제공된 교육용 예제 실습본을 복사해 실행해 본 것에 머무르고, 나만의 진짜 업무 문제를 대입해 개선하려는 시도를 멈추는 것.

스스로 설정한 한계와 고정관념을 직접 깨고, 현업의 치열한 아이디어를 투영해 가치 있는 결과물을 직접 배포해 준 참가자 여러분의 적극적인 태도 덕분에 본 모임은 가장 이상적인 성공의 기준을 달성하며 마무리되었습니다.

2. 변화하는 시대의 미래 인재상

💡 "일의 미래는 학습이다 (The future of work is learning)"

글로벌 최대 규모의 HR 컨퍼런스인 Unleash America 2026의 핵심 메시지 중 하나로, 불확실성이 극대화되는 비즈니스 환경에서 앞으로 끊임없이 배우고 채우는 학습 능력이 중요하다는 것을 강조했습니다.

  • Never Normal (네버 노멀) 시대의 적응: 각종 지정학적 리스크와 AX 기술 진화 등 불확실성이 일상이 된 비즈니스 전장에서는 미리 완벽한 설계를 고집하기 어렵습니다.
  • 빠른 실패와 빠른 학습 (Fast Learner): 완벽하게 짜인 계획서보다, 한발 앞서 유연하게 환경 변화를 인지하고, 완벽하지 않더라도 먼저 빠르게 실행하여 부딪치고, 그 과정 속의 실패를 통해 다시 민첩하게 복기하고 배워 일어나는 사람이 미래 비즈니스를 개척하는 진정한 핵심 인재입니다.
